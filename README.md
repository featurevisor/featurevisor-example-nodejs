# featurevisor-example-nodejs

An example Node.js app using [Featurevisor SDK](https://featurevisor.com).

## Requirements

See required Node.js version in [`.nvmrc`](./.nvmrc) file.

## Installation

```
$ npm ci
```

## Running the app

```
$ node ./src/index.js
```
